Prérequis, objectifs et modules suivants
Compétence à acquérir durant ce module 117
A la fin de ce module vous serez capable d'installer un réseau local (sans administration centralisée des utilisateurs).
Vous serez capable de planifier l'installation d'un nouveau réseau local comportant jusqu'à dix postes de travail (avec accès Internet) et des imprimantes répartis dans différents locaux d'un même bâtiment
Enfin, vous serez capable d'installer les composants réseau à partir de la prise LAN (ordinateur, imprimante locale).

Vous trouverez les objectifs et les compétences opérationnelles en vous rendant sur la fiche d'identification du module 117 et davantage d'informations organisationnelles et techniques sur le descriptif du prestataire.


Les modules précédant le 117
Avant ce module 117, vous avez complété les modules suivants, dont les compétences sont nécessaires :

100 - Distinguer, préparer et évaluer des données (en particulier le binaire et l'hexadécimal)
301 - Appliquer les outils bureautiques (pour la rédaction efficace de votre RP)
Vous complétez aussi le module système suivant :

304 - Installer et configurer un ordinateur mono-poste
Les modules qui suivront le 117
Plusieurs futurs modules de votre formation s'appuieront sur les connaissances que vous développerez durant ce module 117 :

R10 - Mettre en place des systèmes d’interconnexion d’un réseau local
129 - Mettre en service des composants réseau
146 - Relier une entreprise à Internet
184 - Implémenter la sécurité réseau
145 - Exploiter et étendre un réseau
Les logiciels nécessaires pour travailler chez vous, en autonomie
Ces logiciels sont mis à disposition par l'EMF, tant pour une utilisation à l'école que chez vous, téléchargez-les (à l'EMF) pour vous préparer un environnement de travail personnel (à la maison) :

Cisco Packet Tracer : Permettant de simuler et de tester une configuration avant un déploiement
Microsoft Office 365 : Utilisation de la suite bureautique Office notamment Word, Excel et PowerPoint
Microsoft Visio : pour la réalisation des schémas réseau logique et physique.
VMware Workstation Pro : pour pouvoir créer, modifier tant des machines virtuelles que vos réseaux virtuels (la clé de licence annuelle est disponible sur le partage S:\EMF\ProgramSource\vmware\workstation\licence.txt).
Images d'installation du client MS Windows 10/11 et du serveur MS Windows Serveur 2022
Vous pouvez aussi demander à votre enseignant si une machine virtuelle ou un modèle est disponible spécifiquement pour ce module.


Le modèle OSI - Open Systems Interconnexion
Le modèle OSI, permet de diviser les problématiques liées à la communication d'information entre deux points (émetteur - récepteur). Ce chapitre permet de s'approprier ce modèle théorique

Vous pouvez retrouver les informations détaillées de ce modèle OSI sur le site de l'ISO (l'organisation internationale de normalisation)
